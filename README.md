# PuzlPlace

This project runs with Laravel 10, PHP 8.2, Redis and Mysql.

# Installation

## Go to PuzlPlace directory
```shell
cd PuzlPlace
```

## create .env file
```shell
cp .env.example .env
```

## start containers
```shell
$ docker-compose up -d --build
```

Then open in browser:
[localhost](http://localhost)
